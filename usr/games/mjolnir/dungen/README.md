Delaunay Triangulation Dungeon Generator.

    >> Inspired by Tiny Keep's dungeon generator.
    
    >> Slightly revamped from https://github.com/glouw/dungen.git

Use case:

    #include "map.h"

    int main(int argc, char *argv[])
    {
        const struct map        map = mapgen(80, 120, 10, 40);
        
        mapprint(map);
        mapclose(map);
    }

Where the map generation function is defined as:

    struct map mapgen(number of columns,
                      number of rows,
                      size of grid,
                      max number of rooms);

Generated map is stored as a 2D array of characters.
